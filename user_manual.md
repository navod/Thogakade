## Instructions

**Init database**

    db init

**Create items**

    item create <name> <price> <selling_price> <qty>

**Get all items**

    item all

**Get items by id**

    item view <id>

**delete items**

    item delete <id>

**Create customers**

    customer create <name> <address> <contact>

**Get all customers**

    customer all

**Get customer by id**

    customer view <id>

**Delete customer**

    customer delete <id>
